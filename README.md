# API Shop

### Cài đặt


```sh
$ cd API-Shop
$ npm install -d
$ npm start
```

### Tính năng
- Đăng ký tài khoản
- Đăng nhập bằng token.
- Quên mật khẩu ( gửi về mail).
- Cập nhật mật khẩu.
- Thêm sản phẩm theo thư mục.
- Upload file
- Phân quyền Admin, Mod, Member.
- GET sản phẩm, phân trang.
- Sửa, Xóa sản phẩm, thư mục.
- ...
